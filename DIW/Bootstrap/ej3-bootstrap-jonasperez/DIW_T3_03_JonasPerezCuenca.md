## TEORÍA
- **Navbars**: cabecera de navegación versátil y adaptable que organiza los enlaces esenciales de tu web.

- **Cards**: contenedor de contenido flexible y extensible que se utiliza para agrupar información relacionada de manera visualmente atractiva y fácil de gestionar.

- **Botón**: elemento de interfaz interactivo (puede ser una etiqueta button, a, o input) al que se le aplica la clase base .btn.

- **Grupo de botones**: Contenedor que utiliza la clase .btn-group para unir varios botones relacionados en una única línea compacta (horizontalmente).

- **List Groups**: componente muy flexible que se utiliza para mostrar una serie de contenidos o elementos de forma consecutiva, organizados como una lista, pero con estilos y utilidades avanzadas.

- **Tables**: Forma de Bootstrap para manejar tablas con la clase .table.

- **Formularios basicos**: Colección de clases prediseñadas para darles a los formularios una apariencia moderna y responsiva.
